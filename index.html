About PmaYl

This is a simple email program that I made in python. It is currently in development and does not work. In addition, please note that this is not the full documentation. For that, either click here or visit our Docs section.

Please Note:

This program has currently been tested with only gmail. It will likely not work with other providers.
This software has not been tested for security, DO NOT use it for any confidential or personal. This project was only created to learn a bit about software development.
It can not view mail, this software can only send it.
No passwords are stored. While this is a feature I would like to bring in, I need to figure out a way of doing it without using plain text.

About PmaYl

PmaYl started off as a very simple project. Orignally as a part of a motion sensing camera I attempted to make using a Raspberry Pi (which I still haven't completed) that would email you the image once motion had been detected. This then evolved into a school project called "PYMail" where I made a command-line program that would send you emails. As I started to play around with TKinter, I decided to make this project use a GUI and changed the name to "PmaYl" to be slightly more original. And that is where we are today.

I really hope you enjoy my little program

Best Wishes, zeddytbr (Tom)

How To Use PmaYl

Run "start-pymayl.py".
Enter your email and password.
Press Login.
Fill out the recipient, subject and main body form. (Attatchments are not yet supported.)
Hit Send. (This does not currently work, please wait until release.)

Our Future Plans

CCurrently the design is unattractive, I would like to change from TKinter to another GUI option. Likely electron with a python backend. This will likely change after the first official release.
I would also like to make it clearer wherether login has been successful or not.
It would be good if I could allow people to use it in order to read and manage emails, not just send them. However, I think that is beyond my level at this point in time but could be an addition in the future.

The PmaYl Composer

The PmaYl Composer aims to be as intuitive and simplisic as possible while being feature packed. While it is currently limited, it will soon have many more such as HTML formatting and Attachments. We are also planning on moving away from TKinter in favour of Kivy to allow for a more attractive design.

The Login Screen

Simple and straight to the point. That is what this whole project aims to achieve. While we currently only support Gmail by default without configuring the source code, we will soon add support for other providers.
